Homework Frosting

Quickly create a list of your assignments in Microsoft Teams:
  1. Go to https://teams.microsoft.com
  2. Sign in, if necessary
  3. Click on Assignments in the far left navigation bar
  4. Click the cupcake icon in the top right
  5. Wait about ten seconds and the report will be displayed

Options

To filter out assignments you don't want to see, enter words or phrases, one per line. For example, if you never want to see assignments containing "extra credit" or by the teacher Mr. Macinerny, then type each on its own line.



  

Search text is case-insensitive, which means that "extra credit" and "EXTRA CREDIT" will both work.
