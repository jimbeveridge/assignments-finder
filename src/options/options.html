Homework Frosting

This Chrome Extension allows you to easy create a quick snapshot of your assignments in Microsoft Teams. To create a report:

  1. Go to https://teams.microsoft.com
  2. Sign in, if necessary
  3. Click on Assignments in the far left navigation bar
  4. Click the cupcake icon in the top right
  5. Wait about ten seconds and the report will be displayed

Options

It's often necessary to filter out some lines that don't apply. For example, sometimes teachers share a class name and they put their name on the assignments that apply to them.

To filter out assignments you don't want to see, enter words or phrases, one per line. For example, if you never want to see anything containing "extra credit" or by the teacher Mr. Macinerny, then enter the following options in the box below. Search text is case-insensitive, which means that "extra credit" and "EXTRA CREDIT" will both work.

  • extra credit
  • Macinerny
